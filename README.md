# Char-Game
项目名称：你好呀，数分

项目描述：本项目是一个小型的字符游戏，结合了当今热门的游戏教育化的思想，将数学分析这一大学学科基础嵌套在有趣又好玩的RPG游戏中，在感受游戏的视觉冲击与听觉冲击的同时，真正学习到有用的东西

项目配置说明：项目测试 平台为Windows10下Microsoft Visual Studio 2017，Windows SDK版本为10.0.15063.0，平台工具集为Visual Studio 2017（v141），字符集为多字节字符集，使用的编码方式为简体中文(GB2312)-代码页 936
